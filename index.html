온라인 담타(Online Damta)는 비흡연자를 위한 실시간 소셜 커뮤니티입니다. 화면을 눌러 가상 전자담배를 피우며 전국 각지의 사람들과 익명으로 실시간 채팅을 나눌 수 있는 한국어 웹 서비스입니다.

온라인 담타란 무엇인가요?

온라인 담타(Online Damta)는 비흡연자를 위한 익명 실시간 채팅 커뮤니티 웹 서비스입니다. 사용자는 화면을 눌러 가상 전자담배 인터랙션을 즐길 수 있으며, 동시에 전국 각지의 다른 사용자들과 회원가입 없이 익명으로 실시간 채팅을 나눌 수 있습니다. 2026년 5월 출시된 한국어 서비스로, 모바일과 데스크톱 모두 지원합니다.

4가지 핵심 기능

  1. 가상 전자담배 인터랙션 — 화면을 길게 누르면 액상이 감소하고 증기가 발생합니다. 오렌지→노랑→초록 그라데이션 게이지 바로 잔량을 실시간 확인 가능합니다.
  2. 실시간 익명 채팅 (WebSocket + Cloudflare Durable Objects) — 회원가입·로그인 없이 즉시 채팅 참여. 접속 시 자동 생성된 귀여운 닉네임이 부여됩니다. 모든 메시지는 세션 종료 후 Cloudflare Durable Object 메모리에서 자동 삭제됩니다.
  3. 모바일 우선 반응형 UI — iPhone, Android, 태블릿, PC 어디서나 터치 한 번으로 조작 가능. PWA 표준 지원으로 홈 화면에 추가 가능합니다.
  4. 100% 무료 & 익명 — 이메일·전화번호·IP 등 일체의 개인정보 수집 없음. 결제 없음. 광고만 표시되며, 모든 기능 무료.

가상 전자담배 인터랙션 상세 가이드

💨 전자담배 모드

가상 vape 모드. 화면을 누르고 있으면 액상이 0.5씩 줄고, 누르는 동안 추가로 0.015/프레임씩 감소합니다. 기기 안쪽에 오렌지→노랑→초록 그라데이션 게이지 바가 표시되어 잔량을 시각적으로 확인 가능합니다. 0이 되면 자동으로 종료됩니다.

  • 액상: 100 단위, 클릭당 0.5 + 누르는 동안 0.015/프레임
  • 게이지: 오렌지→초록 그라데이션
  • 시각 효과: 증기(vape cloud) 파티클

이용 방법 (5단계 가이드)

  1. 웹 브라우저로 https://onlinedamta.pages.dev에 접속합니다. 회원가입·로그인·앱 설치 불필요. Cloudflare 글로벌 CDN으로 어디서나 빠르게 접속됩니다.
  2. 중앙 캔버스를 길게 누르면 가상 전자담배가 활성화됩니다.
  3. 누르고 있는 동안 액상 게이지가 줄어들고 증기가 발생합니다.
  4. 하단 채팅창에 메시지를 입력해 다른 사용자들과 실시간으로 대화합니다. 한국어, 영어, 이모지 모두 사용 가능.
  5. 액상이 0이 되면 "한대더?"로 다시 시작하거나 "이제 들어갈까?"로 종료를 선택합니다.

자주 묻는 질문 (FAQ)

Q1. 회원가입이 필요한가요?

아닙니다. 온라인 담타는 완전 익명 서비스로, 회원가입·로그인·이메일 인증 없이 브라우저만 열면 바로 이용할 수 있습니다. 접속 즉시 자동 생성된 닉네임(예: 산책하는 고양이, 명상하는 호랑이 등)이 부여됩니다.

Q2. 무료인가요?

네, 100% 무료입니다. 결제가 필요 없고, 숨겨진 비용도 없습니다. 가상 전자담배 인터랙션과 실시간 채팅 기능을 무제한 이용할 수 있습니다. 서비스 운영을 위해 Google AdSense 광고가 표시될 수 있습니다.

Q3. 어떤 기기에서 사용할 수 있나요?

웹 브라우저가 있는 모든 기기에서 사용 가능합니다: iPhone(iOS Safari), Android(Chrome), iPad, 태블릿, Windows PC, Mac, Linux. 모바일 환경에 최적화되어 터치 한 번으로 조작할 수 있습니다. 별도의 앱 설치가 필요 없습니다.

Q4. 채팅은 정말 익명인가요? 개인정보는 안전한가요?

네, 완전 익명입니다. 이메일, 전화번호, 실명, IP 주소 등 일체의 개인정보를 수집하지 않습니다. 채팅에 표시되는 닉네임만 자동 생성된 가명입니다. 모든 채팅 메시지는 세션 종료 후 서버에서 자동 삭제되며, 영구 저장되지 않습니다. 자세한 내용은 개인정보처리방침을 참조하세요.

Q5. 실제 전자담배를 사용하는 건가요? 건강에 영향이 있나요?

아닙니다. 100% 디지털 가상 시뮬레이션입니다. 실제 전자담배, 니코틴, 프로필렌글리콜, 식물성 글리세린, 향료 등 어떤 유해 물질도 없습니다. 비흡연자, 임산부, 미성년자 모두 안심하고 즐길 수 있는 안전한 디지털 서비스입니다. 흡연을 권장하거나 장려하지 않습니다.

Q6. 어떻게 사용하나요?

중앙 캔버스를 길게 누르고 있으면 가상 전자담배에서 증기가 발생하고, 액상 게이지가 오렌지→노랑→초록 그라데이션으로 표시됩니다. 액상이 0이 되면 세션이 종료되고, '한대더?'로 다시 시작하거나 '이제 들어갈까?'로 종료할 수 있습니다.

Q7. 채팅에서 욕설이나 부적절한 메시지를 보면 어떻게 하나요?

온라인 담타는 모든 이용자가 편안하게 소통할 수 있는 공간을 지향합니다. 욕설, 혐오 발언, 광고, 도배, 성희롱 등은 자제해 주시기 바랍니다. 부적절한 행위가 지속되면 서비스 개선을 위해 익명 신고 기능을 추가할 예정입니다.

커뮤니티 가이드라인

온라인 담타는 모든 이용자가 편안하게 소통할 수 있는 열린 공간을 지향합니다. 타인에 대한 존중과 예의를 지켜주시고, 욕설·혐오 발언·광고·도배·성희롱 등은 자제해 주시기 바랍니다. 우리 모두에게 즐거운 커뮤니티를 만들기 위해 서로 배려하는 마음을 지켜주세요.

기술 스택

  • 프론트엔드: HTML5 Canvas, Vanilla JavaScript, CSS3
  • 실시간 통신: 네이티브 WebSocket API (Cloudflare Durable Objects)
  • 호스팅: Cloudflare Pages (무료 티어, 글로벌 엣지)
  • 서버리스: Cloudflare Workers + Durable Objects (WebSocket 중계)
  • SSL/TLS: Cloudflare 자동 (HTTPS, TLS 1.3)
  • DDoS 방어: Cloudflare 무료
  • PWA: 모바일 홈 화면 추가 지원
