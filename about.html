💨 비흡연자도 함께하는 100% 익명 실시간 채팅 커뮤니티

최종 업데이트: 2026-06-08

서비스 개요

온라인 담타(Online Damta)는 2026년 5월 런칭한 한국어 기반의 익명 실시간 채팅 커뮤니티입니다. 화면을 눌러 가상 전자담배(💨 Vape) 인터랙션을 즐기면서 전국 각지의 사람들과 부담 없이 소통할 수 있는 공간을 제공합니다.

회원가입, 로그인, 앱 설치, 결제 없이 브라우저만 있으면 누구나 즉시 참여할 수 있습니다. 모든 채팅 메시지는 세션 종료 후 자동 삭제되어 프라이버시를 보장합니다.

비전과 가치

  • 익명성: 개인정보 수집 없이 자유롭게 소통
  • 안전함: 비흡연자, 미성년자 모두 부담 없이 즐길 수 있는 디지털 경험
  • 무료: 모든 기능을 광고로만 운영, 완전 무료
  • 접근성: 모바일·데스크톱 어디서나 즉시 접속

주요 기능

  • 가상 전자담배 시뮬레이션 (액상 게이지, 증기 이펙트)
  • 익명 실시간 채팅 (랜덤 닉네임 자동 생성)
  • 세션 타이머 (완료 시 M:SS 표시)
  • PWA 지원 (홈 화면 추가)

기술 스택

  • 프론트엔드: HTML5 Canvas, Vanilla JavaScript, CSS3
  • 호스팅: Cloudflare Pages (무료 티어, 글로벌 CDN)
  • 실시간 통신: Cloudflare Workers + Durable Objects (WebSocket)
  • 보안: Cloudflare SSL/TLS 자동 적용, DDoS 방어

면책 고지

온라인 담타는 실제 흡연이나 전자담배 사용을 장려하지 않습니다. 모든 시각적 요소와 인터랙션은 순수한 디지털 시뮬레이션입니다. 서비스는 개인 오픈소스 프로젝트로 운영되며, 특정 집단이나 상업적 이해관계와 무관합니다.
